### Run Instructions

1. Clone the repository

2. Create a virtual environment 
### For Windows
```bash
python -m venv venv
```
### For Linux
```bash
python3 -m venv venv
```
3. Activate the virtual environment
### For Windows
```bash
venv\Scripts\activate
```
### For Linux
```bash
source venv/bin/activate
```
4. Install the requirements
```bash
pip install -r requirements.txt
```
5. Run the app
```bash
python app.py
```
